All Important Git Commands

Manoj Kumar 16 August 2021 comments

Wellcome to MyBlog in this article I will discuss about all important git Command and it's uses. So let's Start the topic

Following are the Imoprtant GIt Command that frequently used by the developers

  1. Configuration

    Configure your Git username and email using the following commands, replacing “username” with your own. These details will be associated with any commits that you create:

                    
                    $ git config --global user.name "username"
                    $ git config --global user.email "

  2. Initialize GIT local repository: (Repository is initialized only once at the time of configuring VCS)

                    
                    $ git init

  3. To check the status of tracked/untracked files and folders:

                    
                    $ git status

  4. To track files and folders which are yet to commit or staging:

                    
                    $ git add .  # to add all changes

                    $ git add path/to/file  # to add particular file

  5. To commit changes:

                    
                    $ git commit -m "commit message"

  6. To add Remote Origin to add committed files and folder to server repository:

                    
                    $ git remote add origin online_repo_URL

                    Example
                    # git remote add origin https://github.com/username/test-git.git

  7. To add committed files and folders to the master branch:

                    
                    $ git push -u origin master

  8. To see the list of branches:

                    
                    $ git branch

  9. To create new branch:

                    
                    $ git branch 'branch-name'

                    Example

                    $ git branch newbranch

  10. To checkout to specific branch:

                    
                    $ git checkout 'branch-name'

                    Example

                    $ git checkout newbranch

  11. To Merge NewBranch to Master:

                    
                    $ git merge 'branch-name'

                    Example 

                    $ git merge newbranch

                    # Make sure You are in master barnch

  12. To Delete branch from Local Repository

                    
                    $ git branch -d 'barnch-name'

                    Example

                    $ git branch -d newbranch

  13. To Delete branch from Remote Repository:

                    
                    $ git push origin --delete 'branch-name'

                    Example

                    $ git push origin --delete newbranch

  14. To clone the git repository:

                    
                    $ git clone 'repository url'

                    #Example

                    $git clone https://github.com/username/test-git.git

  15. To pull the changes:

                    
                    $ git pull

                    $ git pull origin 'branch-name' # To pull specific branch

                    Example

                    $ git pull origin newbranch
