What is Git?

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Git is easy to learn and has a tiny footprint with lightning fast performance.

VERSION CONTROL SYSTEM(VCS)

Version control systems (VCS) are a type of revision control system. A revision control system maintains a history of changes to a file or set of files.

Version control systems provide developers with the ability to track changes to files, and revert changes or "roll back" to earlier versions. Version control systems also provide the ability to revert changes to files in case of an error or a problem.

Version control systems are used in software development, especially in software engineering. Version control systems are used in many different domains, including computer science, engineering, and law. Version control systems are also used in the design and construction of software products, such as operating systems, computer software, and web sites.

Following are two types of Version Control Systems.

  1. Centralized Version Control System (CVS)

    CVS is a distributed version control system that tracks changes to files in a central repository. CVS is often used to track changes to computer programs, but can be used for any type of project. CVS is often used as a backup system for projects that are already using version control systems.

  2. Distributed Version Control System (DVCS)

    DVCS is a distributed version control system that tracks changes to files in a distributed repository. DVCS is often used to track changes to computer programs, but can be used for any type of project. DVCS is often used as a backup system for projects that are already using version control systems.
